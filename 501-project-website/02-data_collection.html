2.Data Collection

The overview of published datasets

Practitioners in NLP field has conducted numerous researches to deal with hate speech detection. There are plenty of datasets published online related to my research project. My first thought was to employ one of them to conduct my research project. However, the first reason why I decided not using them is because most of the researchers only leveraged the text data to conduct the NLP techniques. I believe by conducting the network analysis, and also combining it with traditional ML techniques, I am capable of achieving this speech violence detection task better. Besides, the datasets published online are not the latest. Fancy expression of hate speech emerge for years to come.

Twitter is widely used throughout the world. Much as there is a large-scaled layoffs reported by News in the recent months, few can shake its hegemony in social media. By using tweepy, which is a official API for developer of tweeter, I can leverage the tweepy to scrape tweets and download them to analyze the text data. Tweepy API enables me to collect both users' portfolio information and rich text information from tweets.

I mostly use two datasets throughout this research project. The first one is a large dataset generated by Waseem, Zeerak and Hovy, Dirk,which have been annotated for hate speech. It contains 16900 rows and two variables. The original purpose is for NAACL Student Research Workshop.

I downloaded all the data sets that I might need in the research. There are three in general, namely, NAACL_SRW_2016.csv, twitter_en.csv and cyberbullying_tweets(Kaggle).csv. The Kaggle data set is not used in this research project because the lack of numerical variables. Last, the hate_speech.csv data set is the cleaned format of the NAACL_SRW_2016.csv data set.

I lookuped each tweet ID using the Tweepy API so as to collect all the data I need. Then the data can be divided into numerical variables and text variable. Numerical variables include Followers numbers, friends numbers, retweet numbers, favorite numbers, id. The id variable represents the timestamp that generate for each tweet. It is unique and meaningful that should not be discarded. Text variable represent the text data of tweets, which is the most significant variable to predict hate speech.
